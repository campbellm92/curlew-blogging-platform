User guide

To use Curlew, you'll want to have basic familiarity with the following technologies:

  • Visual Studio Code: most basically for writing the .md files for blog posts.
  • Markdown: for writing blog posts.
  • Enough HTML to remove/change in the platform release you've downloaded.
  • Enough CSS to change the styling if you want to.

If that doesn't sound too bad, follow these steps:

  1. Download the latest version:
    Visit the releases page on GitHub to download the latest version of the app.
  2. Unzip the folder:
    Unzip the folder wherever you'd like the app installed/wherever your working directory will be. Rename it if you want to.
  3. Open the project with VS Code:
    Open the unzipped folder in VS Code. You can do this by right clicking on the folder and selecting 'Open with VS Code', or simply by going to File > Open Folder in VSCode.
  4. Install dependencies:
    Go to the top menu and open a new terminal (Terminal > New Terminal). In the terminal window that appears, run 'npm install' to install the necessary dependencies.

From there, you're basically good to go. You can add posts in the 'posts' directory with the file extension '.md'. Writing Markdown is pretty simple, and you can find a guide to the syntax here. You can also feel free to remove any HTML pages you don't want and change the styling in CSS.

If you'd like to find out how you can deploy your blog to an actual live domain, visit the deploy page.

Happy blogging!
