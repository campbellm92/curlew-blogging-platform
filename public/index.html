This is Curlew: a simple, lightweight blogging platform. It is intended to be used by developers, but not necessarily exclusively.

With Curlew, you write blog posts as markdown (.md) files which you save in the 'posts' directory. Once you've saved your work, the app takes care of converting it into HTML, which is what your reader sees in their browser. You can see an example in posts.

Why use Curlew?

  • It utilises a minimal aesthetic which is highly customisable.
  • It's already optimised to look good on all devices.
  • It's lightweight with minimal dependencies.
  • It's configured for syntax highlighting for if you want to include code blocks in your blog posts.
  • It's easy to deploy.

Please visit the guide for a complete walkthrough on how to use Curlew and deploy to find out how to deploy your blog with heroku and a domain provider of your choice.

I'm personally using a version of curlew for my personal dev portfolio site.
